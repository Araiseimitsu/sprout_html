Overview

バランスの極致

GLM-5.2 は高速な応答性と深い推論能力を両立し、コード生成・文章作成・多言語理解のいずれにおいても卓越したパフォーマンスを発揮します。

01

深い推論

段階的思考（Chain-of-Thought）を内蔵し、複雑な問題も論理的に分解して解答します。

02

高速応答

最適化された推論エンジンにより、低レイテンシでストリーミング出力を実現します。

03

コード生成

20以上のプログラミング言語に対応。リファクタリング・デバッグ・テスト作成まで。

04

多言語対応

日本語を含む自然な多言語対話と、文化的な文脈を踏まえた回答が可能です。

Features

見せる、変わる、応える

単なるテキスト生成を超えた、インタラクティブで文脈を把握する知能。

動的知能コア

入力の意図を即座に解析し、最適な推論パスを動的に選択。質問の裏にある「本当に知りたいこと」を汲み取ります。

並列ストリーミング

トークンを並列生成し、最初の1文字までの時間を最小化。長文でも待たせない、途切れない会話体験。

文脈グリッド記憶

広大なコンテキスト窓を持ち、会話の流れ・ファイル全体・ドキュメント群を一度に保持。忘れない、見失わない。

# GLM-5.2 が生成: カメラ映像の差分検知 (1fps)
import cv2, time, os

def detect_motion(cam=0, fps=1, threshold=0.015):
   cap = cv2.VideoCapture(cam)
   prev = None
   while True:
      ok, frame = cap.read()
      if not ok: continue
      gray = cv2.GaussianBlur(cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY), (21,21), 0)
      if prev is not None:
         diff = cv2.absdiff(prev, gray)
         if (diff > 25).mean() > threshold:
            cv2.imwrite(f'motion_{time.time():.0f}.jpg', frame)
      prev = gray
      time.sleep(1/fps)
